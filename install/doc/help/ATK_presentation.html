ATK Presentation

The Accelerator Test Kit is a software that emulates the user interaction with a phone, allowing automatic testing of embedded applications.

During a test, ATK measures device resources consumption (cpu, power, memory etc ...) in real-time, and then provides graph report (pdf and html format) as a result. The ATK "Graph Analyzer" tool enables you to map together different sources of data graph and events in order to perform correlation.

Connect the device you want to test with USB cable. Then launch ATK Test Manager : select Start > Programs > ATK > ATK. You may have to click on the "Device detection" button to select the correct device.

There are three tabs in ATK Test Manager :
  • Script Test tab
    • , that enables you to load different script tests and launch them sequentially on the device.
  • Random Test tab
    • , that enables you to perform random test ("monkey" test) on the device.
  • MixScript Test tab
    • , that enables you to load different script tests and launch them in a random order on the device.



The ATK "Script Recorder" tool enables you to record actions performed on the mobile device (touch screen, key press ...). These actions are translated into script language commands, and saved into test script file. Other script commands can be manually added to the script to perform special actions such as taking screenshot, make a loop on a set of actions, including another script file etc...

As mentionned above, ATK is also able to take screenshots during a test. These screenshots are used to check the application behaviour or state during the test. To avoid fastidious manual check, the ATK "Screenshot comparator" automatically compares the resulting set of screenshots with a set of reference screenshots, and points out the failing images.
