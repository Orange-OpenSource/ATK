Monitoring Android Phones

Here are the various ressources ATK is able to monitor on Android Phones.

Storage

Storage usage : number of KBytes of used internal storage (ROM).

DSCard

External storage usage : number of KBytes of used external storage (SDCard).

Memory

Memory usage : number of KBytes of used memory (RAM).

CPU

CPU usage : percentage of CPU load, which represents the % of CPU occupation * by the % of CPU frequency.

Battery

Battery level : percentage of battery charge.

Data sent

Data sent : amount of data sent (between two measures).

Data received

Data received : amount of data received (between two measures).

Process

You can also monitor CPU usage and Memory usage for a specific process. See Monitoring configuration section to understand how to monitor process.
Please note that ATK will provide the list of process that can run on the phone. It does not guarantee that the specified process will be running during the test. If a process is not running, its ressource consumption graph will be empty.
