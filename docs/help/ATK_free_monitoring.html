ATK Free monitoring

A new feature has been added to enable user to monitor its phone resources without running any script.

Launch ATK Free monitoring

Connect the device you want to monitor with USB cable. Then launch ATK Test Manager : select Start > Programs > ATK > ATK. You may have to click on the "Device detection" button to select the correct device.




Click on the button. A new window will appear :




Select a configuration file. This will define which resources (cpu, memory, storage ...) or process will be measured, and what will be displayed in the resulting graphs report. See Monitoring configuration section to understand how to configure your tests with the wizard.
  • Press "Start" button to start monitoring the resources of your phone. You may have to specify the target directory for the results of the monitoring.
  • Press "Stop" button to stop the monitoring : a report will be automatically created and stored in a new folder called "Manual_<Current_Date_Time>" that you'll find in the result directory you have specified.
  • Press "View Report" if you want to see the last report generated
  • Press "Cancel" button to stop the monitoring and not generate any report.
  • Press "Graph Analyser" button to see the last generated graph analyzer.

Url markers

Url markers are available in real time graphics if you have activated "Network Monitor" in the settings (if your phone is rooted and has busybox installed). It will show you what web site has been requested and when.
You can choose to show/hide the markers by clicking the "Display Url markers" checkbox.
