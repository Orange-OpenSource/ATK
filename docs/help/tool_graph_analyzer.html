ATK Graph Analyzer

The aim of the Graph Analyser tool is to map together different sources of data graph and events in order to perform correlation.

In order to open the ATK Graph Analyser tool, click on the graph icon in the " ATK Test Manager" tests check-list, or select Start > Programs > ATK > Analyzer.



If you open the Graph Analyzer within the ATK Test Mananager, it will display the resulting graphs of the selected test.

If you open the Graph Analyzer from the Start menu, you will have to indicate a test result folder manually : open "File > Open Directory" menu and indicate the path where the test results are stored. You can also add another graph by using "File > add a reference graph" menu, and indicating a specific result .csv file.
