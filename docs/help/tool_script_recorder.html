ATK Script Recorder

This tool can record, play and edit user actions scenarii on a mobile phone. These scenarii are displayed on the screen in a simple script format, and can be saved into .tst test script files.

In order to open the ATK Script Recorder tool, click on the red circle button of the " ATK Test Manager", or select Start > Programs > ATK > ScriptRecorder.

Recording a script

Plug your phone with a USB cable. You may have to click on the "Device detection" to choose the phone on which you want to record actions. In order to record a new script, click on the red circle button :

You can start doing actions on your phone. The script recorder is able to detect key presses on all the phones and touch presses, touch slides and touch drag & drops on touchscreen phones. When you have finished, click on the blue square button to stop recording. You can see the recorded script commands on the recorder window.

If you want to take a screenshot of your phone during the scenario, click on the photo camera button. Indicate the folder were the screenshots will be saved during the current recording session.

Editing a script

You can modify the current script to add, delete or modify action command lines. Just select one or several lines and right click to make the submenu appear:

This submenu enables you to perform some actions on the selected lines :
  • Delete Selected lines
    • , to remove the selected lines from the current script,
  • Surround with Loop
    • , to surround the selected lines with a Loop command,
  • Comment/Uncomment lines
    • , to comment or uncomment the selected lines,
  • Insert line Before
    • , to insert a new line just before the selected lines,
  • Insert line after
    • , to insert a new line just after the selected lines.
When you insert a new command line, the recorder perfoms auto-completion. It will automatically propose a list of commands that can complete the command your are typing :

Saving a script

Once your script is finished, you can save it into a .tst file. In the "File" menu, select "Save as..." and indicate the path where the file will be saved.
If you want to create a new script, go to the "File" menu and select "New".

Loading a script

Open the "File" menu and select "Open...", then indicate the path of the .tst script file you want to load in the script recorder. Once loaded, the script command lines must appear on the recoder screen.

Playing script

To start playing a script on the connected phone, select the blue arrow button. The script command currently played is displayed in bold:


This will enable you to check if the script is correctly played on the phone. At any time, you can click on the blue square button to stop script playing.

Note that script recorder does not log resources consumption when playing. To log resources you must use ATK main tool.
