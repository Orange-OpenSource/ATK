ATK Random tests

The ATK Test Manager "Random Test" tab enables you to perform random testing (also called "monkey" testing) of the applications installed on your phone.

Launch ATK Test Manager

Connect the device you want to test with USB cable. Then launch ATK Test Manager : select Start > Programs > ATK > ATK. You may have to click on the "Device detection" button to select the correct device. Select the "Random Test" tab.




Adding scripts to the check-list

Click on the button, to make appear the list of avaible applications on your device. If ATK can't find any application automatically, it will let user enter manually the application name. Select one or multiple applications that you want to test :



For more details on how to manage test check-lists and configurations see ATK script tests section, which is quite analoguous (except we do not manipulate scripts files here).

Running random tests





The name of the tested applications is used as Random Test name. Select your Monitoring configuration and indicate the number of events to perform during the test (or time in sec for Nokia phones). Click on button to launch the random tests.

Viewing random tests results

In order to view the results of a test, double-click on its "Verdict" case, or select the test line and click on the button.

You can launch the Graph analyzer tool, to view the resource consumption graphs (the resulting graph datas are stored under <result_folder>/<your_random_test_name_folder>).
